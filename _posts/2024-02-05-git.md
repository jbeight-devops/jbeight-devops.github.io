# Git

## 1. intro
- git 활용에 대한 기록
### config
- Local 초기 설정
~~~
git config --global user.email "@"
git config --global user.name "jbpark"
~~~

## 2. Branch

- branch 생성
~~~
git branch develop main
~~~

- 현재 작업 중인 branch 확인
~~~
git branch
~~~

### Rules
- 내부적으로 명명 규칙을 정하고, 미사용 branch는 삭제하도록 한다.
- branch 명명 규칙 (예)
  - main
  - develop
  - hotfix(긴급한 문제 해결용)
